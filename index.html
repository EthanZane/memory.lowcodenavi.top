About the Memory Test

What is Memory Test Challenge?

Our Memory Test Challenge is designed to help you improve your short-term memory capacity through an engaging and progressive number memorization game. Whether you're a student looking to enhance your study skills or someone interested in maintaining cognitive fitness, this game offers a fun way to challenge yourself.

How to Play

Game Rules

  • The game consists of 20 progressive levels
  • Each level displays a number for 5 seconds
  • Numbers increase from 1 to 20 digits
  • Memorize and input the number correctly

Scoring System

  • Points per level = Level number × 5
  • Level 1: 5 points
  • Level 20: 100 points
  • Maximum total score: 1050 points

Performance Levels

  • Excellent: 840+ points
  • Good: 630-839 points
  • Average: 420-629 points
  • Need Practice: Below 420 points
